Getting started creating a Tōku chat

Disclaimer; Only verified users can create chats on Tōku

To start please email us the following info, the name of your chat/the description, your username and @, a 100 by 100
pixels icon. This will be displayed on the Tōku home page. Make sure the icon is original and it clearly displays the topic
of your discussion. You may also ask for a darkmode page if you want, but its not required. Remember: All basic Tōku
rules apply, discussions will be closed if necessary. Once you have provided the info we will try to add your chat as soon
as posible.
