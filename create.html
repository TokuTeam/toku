Getting started creating a Tōku chat

Disclaimer; Only verified users can create chats on Tōku

Creating a chat on Tōku can help you organise your group or club, and getting started is easy! To start please gather the
following info.

A name and description for your chat

Your username and @

A 100-100 pixels icon. This will be displayed on the Tōku home page. Make sure the icon is original and it clearly displays the topic of your discussion.

Remember: All basic Tōku rules apply, discussions will be closed if necessary. Once you have provided the info we will try to add your chat as soon as posible.
